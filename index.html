Utilisation des lignes de commande sous linux :

Commandes de base pour ubuntu :

Passer en super utilisateur avec la commande « sudo su » vous permet de passer outre le fait de taper sudo à chaque début de ligne de commandes.

Apt-get et la gestion de paquets :

Comme dit juste au-dessus si vous êtes en super user, vous n'aurez plus à taper « sudo » avant chaque ligne de commandes sinon vous devrez taper sudo à chaque début de ligne.

  • Apt-get update : permet la mise jour des paquets disponibles.
  • Apt-get upgrade : met à jour les paquets déjà installés.
  • Apt-get install « nom du fichier » : permet d'installer un logiciel donné.
  • Apt-get remove «  nom du fichier » : désinstalle le logiciel.
  • Apt-get autoremove « nom du fichier » : désinstalle le logiciel et toutes ses dépendances.

Vérification des fichiers et sous-dossiers avec la commande « ls » list :

  • ls : affiche les dossiers.
  • ls -l : permet d'afficher les informations des sous-dossiers.
  • ls -a : affiche les fichiers cachés.
  • ls -al : affiche tous les fichiers même ceux qui sont cachés.

Déplacement dans les dossiers avec la commande « cd » Change directory :

  • cd /home/utilisateur/dossier Permet d'arriver directement au répertoire dossier.
  • cd .. vous permet de remonter au répertoire précedent.

Création de dossier avec « Mkdir » Make directory :

  • Mkdir « nom_du_dossier » vous permettra de créer un dossier voulu.

Création de fichiers

  • touch permet de créer un fichier : touch « nom_du_fichier » + « extension du fichier »
  • cat affiche les fichiers dans un terminal

Suppression de dossier avec « rmdir » remove directory :

  • rmdir « nom_du_dossier » permet de supprimer un sous-dossier.

Exit ou comment sortir du terminal :

Si vous êtes en super user, taper exit vous permettra de quitter le mode administrateur et si vous tapez une deuxième fois exit vous fermerez le terminal.

Les easters eggs dans la fenêtre de commande :

Fenêtre à la matrix en installant cmatrix « apt-get install cmatrix » puis le lancer en tapant cmatrix, faire de même en installant sl avec la même commande et le lancer pour voir un train dans la fenêtre de commande. Afficher une vache en tapant apt-get moo.
